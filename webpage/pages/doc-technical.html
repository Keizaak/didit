Documentation Technique

Comment ça marche ?

Notre solution se base sur l’accès au site web grâce à une puce NFC integrée dans une carte créée au FabLab du Dôme. L’accès au site se fait lorsqu’on approche la carte d’un smartphone qui accepte la technologie NFC. Nous atterissons alors sur la page d’accueil, il suffira par la suite de se connecter pour accéder aux différentes communautés.


Pourquoi ça marche ?

L’hébergment du site web se fait en utilisant la machine virtuelle mise à disposition par le Dôme, où l’installation d’Apache2 est réalisée en utilisant le port 80.


Avec quoi ça marche ?

Langages de programmation : HTML, CSS, JavaScript, PHP
Framework : Bootstrap
IDE : IntellIij, PHPStorm, WinSCP, Xampp, Wamp…
Lien du Gitlab


Prérequis pour l’installation

L’utilisation de l’application requiert un smartphone qui comporte la technologie NFC (mais il faut l’activer dans certains smartphones).


Voies d’améliorations

L’implémentation d’une base de données pour mieux gérer les communautés, ainsi que les comptes des utilisateurs. L’opérabilité d’un scanner de pièces d’identité pour vérifier les informations de l’utilisateur. Inscription à une communauté conditionnée par l’âge de l’utilisateur (défis dangereux pour les jeunes par exemple). Contrôle parental pour les plus jeunes. L’intégration des entreprises en tant que communautés officielles.


Public cible

Notre application tient la majorité de ses caractéristiques des réseaux sociaux, qui ont pour public cible principal les adolescents et les jeunes adultes. Sur TikTok, l’inscription est conditionnée par un âge minimum de 13 ans, pareil pour Snapchat. Pour Facebook en revanche, cela varie en fonction du pays. Nous nous sommes inspirés de ces informations, et avons donc choisi de viser un public de jeunes adultes et d’adolescents ayant au minimum 13 ans.
